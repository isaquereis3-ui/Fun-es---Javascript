# Projeto: TDE 1

## Descrição
Este projeto demonstra o uso de diferentes tipos de funções em JavaScript, incluindo:
- Funções nomeadas
- Funções que manipulam arrays
- Arrow functions
- Funções que recebem outras funções como parâmetro (callbacks)

## Funções criadas

1. **somar(a, b)** – Retorna a soma de dois números.
2. **subtrair(a, b)** – Retorna a diferença entre dois números.
3. **multiplicar(a, b)** – Retorna o produto entre dois números.
4. **dividir(a, b)** – Retorna o quociente de dois números.
5. **filtrarPares(numeros)** – Filtra e retorna apenas os números pares de um array.
6. **calcularMedia(numeros)** – Calcula a média dos valores de um array (arrow function).
7. **aplicarOperacao(a, b, operacao)** – Recebe uma função de operação e aplica aos valores (callback).
